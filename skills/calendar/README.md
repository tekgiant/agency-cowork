# Calendar Skill

Manage Outlook calendar events using the `microsoft-outlook-calendar` MCP server — create, update, delete, accept, decline, cancel events, find meeting times, and check availability.

## Capabilities

- Create, update, and delete calendar events
- Accept, decline, tentatively accept, and cancel invitations
- Find available meeting times across attendees
- View calendar events and free/busy schedules
- Support for recurring events and online meetings (Teams)
- Forward events to other recipients

## Registration

Registered as an installed plugin in `~/.copilot/config.json` under `installed_plugins` with `marketplace: "local"` and `cache_path` pointing to the plugin root.
