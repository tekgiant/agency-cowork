# D365 Expense

Create and manage expense reports in Microsoft Dynamics 365 Finance & Operations via Playwright browser automation. Attaches corporate card transactions to new reports with approvers and cost center configuration.

## Features

- Browser automation via Playwright for D365 interaction
- Corporate card (Amex) transaction attachment
- Approver assignment on expense reports
- Cost center selection and configuration
- Itemized hotel expense handling
- Receipt processing and attachment
- Complete review-ready draft generation

## Usage

Trigger phrases: "expense report", "file expense", "submit expense", "expense reimbursement", "D365 expense".

Examples:
- "File an expense report for my recent travel"
- "Submit my corporate card charges from last month"
- "Create an expense report with hotel itemization"

## Prerequisites

- Microsoft Dynamics 365 Finance & Operations access
- Playwright browser automation dependencies installed
- Corporate card transactions visible in D365
