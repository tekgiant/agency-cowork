# Dashboard

Interactive, live-updating knowledgebase dashboard with AI action controls. Renders as an iframe embedded in the Agency Cowork UI, providing a visual overview of knowledge and task status.

## Features

- Live-updating knowledgebase visualization
- AI action controls for common operations
- Customizable layout and widget arrangement
- Iframe-based rendering within the Agency Cowork UI
- Dark and light theme support
- Real-time data refresh

## Usage

Trigger phrases: "dashboard", "show dashboard", "open dashboard", "KB dashboard".

Examples:
- "Open my dashboard"
- "Show the knowledgebase dashboard"
- "Customize the dashboard layout"

## Prerequisites

- Agency Cowork UI running with iframe support
- Knowledgebase populated under `memory/Knowledgebase/`
