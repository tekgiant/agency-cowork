# Email Triage

Automated email triage — categorizes inbox emails into Urgent, Needs Response, FYI, and Archive tiers, drafts responses, filters noise, and delivers summaries to Teams self-chat. Runs on a 30-minute schedule during working hours or on demand.

## Features

- 4-tier categorization: Urgent, Needs Response, FYI, Archive
- Automated response drafting for actionable emails
- Noise filtering to reduce inbox clutter
- Teams self-chat delivery of triage summaries
- 30-minute scheduled runs during working hours
- On-demand invocation for immediate triage
- Sensitive content detection before posting summaries

## Usage

Trigger phrases: "triage my email", "check my inbox", "process my emails", "what needs my attention", "email triage", "categorize emails", "sort my inbox", "inbox zero".

Examples:
- "Triage my email"
- "What needs my attention in my inbox?"
- "Draft responses for urgent emails"

## Prerequisites

- Microsoft Outlook mail MCP configured
- Microsoft Teams MCP configured for self-chat delivery
- Task scheduler running for automated 30-minute cycles
