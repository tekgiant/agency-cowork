# Excel

Create and edit Excel workbooks with configurable branding. Supports sheets, formulas, charts, conditional formatting, and batch editing. Primary workflow is edit-first: download, inspect, batch edit, and upload back.

## Features

- Create new workbooks from scratch
- Edit existing .xlsx files
- Formulas and calculations
- Chart generation and customization
- Conditional formatting rules
- Batch cell editing for bulk operations
- Configurable branding and styling
- Multi-sheet workbook support

## Usage

Trigger phrases: "edit a spreadsheet", "update an Excel file", "modify a workbook", "create an xlsx", "inspect a spreadsheet".

Examples:
- "Create a new Excel file with monthly budget data"
- "Add a chart to the existing sales spreadsheet"
- "Apply conditional formatting to highlight overdue items"

## Prerequisites

- Node.js runtime for workbook generation
- Source .xlsx files accessible for edit operations
