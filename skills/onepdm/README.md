# OnePDM

Search, browse, and download specifications from OnePDM (Aras Innovator PLM). Supports document search by number or keyword, numbered list browsing with filtering, selective download, retrieval tracking, and batch operations from SOW/PRD exhibits.

## Features

- Document search by number or keyword
- Numbered list browsing with filtering
- Selective specification download
- Retrieval tracking for audit purposes
- Batch operations from SOW/PRD exhibits
- Aras Innovator PLM integration
- Spec freshness checking

## Usage

Trigger phrases: "onepdm", "specification", "spec download", "plm", "document number", "download spec", "list specs", "check freshness", "import sow".

Examples:
- "Search OnePDM for specification M1345662"
- "Download all specs referenced in the SOW exhibits"
- "List specs and check freshness"

## Prerequisites

- OnePDM (Aras Innovator) access credentials configured
- Network access to the PLM instance
