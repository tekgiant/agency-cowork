# PowerPoint

Dual-engine PowerPoint skill — uses pptxGenJS for creating new decks with rich visuals and python-pptx for editing existing files with DRM handling. Automatically selects the appropriate engine based on the operation.

## Features

- Create new presentations with pptxGenJS
- Edit existing .pptx files with python-pptx
- DRM-protected file handling for corporate templates
- Rich visual layouts and slide designs
- Speaker notes support
- Slide transitions and animations
- Template-based deck generation
- Brand-consistent styling via DESIGN.md

## Usage

Trigger phrases: "create a PowerPoint", "make a deck", "build a presentation", "edit slides", "update this pptx", "redesign this deck", "add slides".

Examples:
- "Create a presentation for the quarterly review"
- "Edit the existing deck to add a new section"
- "Make me a slide deck about the project roadmap"

## Prerequisites

- Node.js runtime for pptxGenJS engine
- Python >= 3.11 with python-pptx for edit operations
- DESIGN.md (optional) for branded output
