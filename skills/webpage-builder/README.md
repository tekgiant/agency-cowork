# Cinematic Landing Page Builder

Build high-fidelity, cinematic landing pages from 4 simple questions. Every site feels like a digital instrument — intentional scroll, weighted animations, zero generic AI patterns.

**Stack:** React 19 + Tailwind CSS 3.4 + GSAP 3 (ScrollTrigger) + Lucide React + Vite

## Quick Start

Tell the agent: "Build me a landing page" — it will ask 4 questions, then scaffold and build the entire site.

## What Gets Built

1. **Floating Island Navbar** — morphs from transparent to frosted glass on scroll
2. **Cinematic Hero** — full-bleed Unsplash imagery with GSAP stagger animations
3. **Interactive Feature Cards** — Diagnostic Shuffler, Telemetry Typewriter, Cursor Protocol Scheduler
4. **Manifesto Section** — parallax texture with word-by-word scroll reveal
5. **Sticky Stacking Protocol** — pinned cards with SVG canvas animations
6. **Pricing/CTA Grid** — three-tier layout with highlighted middle card
7. **Textured Footer** — system status indicator, rounded top corners

See `SKILL.md` for the complete specification.
