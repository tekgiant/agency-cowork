# Word Doc Skill

Create, read, and comment on Microsoft Word documents via OneDrive and SharePoint using the `microsoft-word` MCP server.

## Capabilities

- Create new Word documents in OneDrive (HTML or plain text)
- Read document content and metadata from SharePoint/OneDrive URLs
- Add comments to documents
- Reply to existing comment threads

## Registration

Registered as an installed plugin in `~/.copilot/config.json` under `installed_plugins` with `marketplace: "local"` and `cache_path` pointing to the plugin root.
